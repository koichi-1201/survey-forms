SharePoint 接続設定

設定方法：
1. Azure Portal で アプリの登録 を行い、Client ID を取得してください。
2. 権限に Sites.ReadWrite.All（または Sites.Manage.All）を付与します。
3. Redirect URI に このファイルを配置したSharePointページのURLを設定します。
4. 対象の SharePoint サイトの URL と、データを保存する List 名を入力してください。
※ M365 Business Standard では Azure AD アプリ登録は無料で利用できます。

ここに記載されたアカウントのみマイナンバーが表示されます（M365 ログイン連携）

アンケート回答フォーム配信設定（GitHub Pages）

アンケートの回答フォームはSharePointではなくGitHub Pagesで配信されます（SharePointの仕様によりHTMLが正しく表示されないため）。事前にGitHubアカウント・公開リポジトリ・Personal Access Tokenの準備が必要です。

repo権限を持つトークンを入力してください（Settings → Developer settings → Personal access tokensで発行）

有効期限を設定しておくと、期限が近づいた際にツール起動時に警告が表示されます

権限管理

ロール説明：
🔴 管理者（Admin）：全機能利用可。登録・編集・削除・設定変更・権限管理
🟡 編集者（Editor）：登録・編集のみ。削除・設定変更・権限管理は不可
🟢 閲覧者（Viewer）：閲覧のみ。一切の編集・登録・削除不可
※ どのロールにも含まれないアカウントは自動的に閲覧者になります
